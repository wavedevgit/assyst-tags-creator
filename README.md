# assyst-tags-creator

Awesome utility to make assyst javascript tags with error handling

## Getting started

First run

```
npm create assyst-tag
```

1. Follow the guide shown in the screen
2. Write your tag
3. Build it and publish using:

```sh
npm run build
npm run publish # pushes minified built source code to github repo
```

4. Now run this command to get an easy to run command on your discord server to add the tag (if you haven't already)

```sh
npm run generate:create
```

5. Enjoy an automatically updating assyt tag!
